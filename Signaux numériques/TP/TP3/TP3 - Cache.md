# Exercice 2
Le degré d’associativité indique combien de lignes un élément donné peut occuper dans un ensemble d’un cache. 

LRU : Last recently used (sert a savoir quel bloc est le plus vieux et remplace ce bloc par la donnée a remplacer (impossible dans un bloc a correspondence directe))


Taille des bloc du cache (une ligne) : $8$ octets
Accès consécutifs :
- $\frac{8}{4} = 2$ pour un pas de $1$
- $\frac{8}{4 \times 2}=1$ pour un pas de $2$
