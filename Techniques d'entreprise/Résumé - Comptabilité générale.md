#### Calcul de l'intérêt
$$I = C \times i \times a$$
$I$ : Intérêt brut
$C$ : le capital demandé
$i$ : le taux d’Intérêt annuel
$a$ : le nombre d'années sur lequel le prêt dure (ce nombre peut être un float)
